Design: remove-schema-system

Decisions

  1. Eliminate schema CLI surface (`c3spec schema ...`).
  2. Remove schema selection knobs from change/workflow commands.
  3. Replace resolver-based artifact wiring with fixed internal workflow contract.
  4. Delete schema directories, including superpowers-bridge.

Migration

  • Keep command outputs/flow equivalent where possible.
  • Update tests/specs to new non-schema contract.

Risks

  • Large test churn
  • Legacy docs drift
