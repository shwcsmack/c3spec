# c3spec Finishing Development Branch

1. Confirm clean tracked state or list remaining tracked changes:
   - `git status --short`
2. Summarize verification evidence (tests/lint/typecheck/build).
3. Prepare commit/PR handoff summary referencing proposal/design/verify/retrospective artifacts.
4. Provide final next actions:
   - commit command(s)
   - push command
   - PR creation suggestion

Do not silently discard changes. If not ready, report blockers clearly.
